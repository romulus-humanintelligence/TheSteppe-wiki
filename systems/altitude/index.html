    Altitude System¶

    Per-tile elevation on local maps affects movement cost, vision range, and visual appearance.

    Elevation Data¶

    Each tile has a float elevation value (0.0-1.0) stored in a parallel array.

    Elevation Bands¶

    Band Range Meters Description
    Lowland < 0.15 150-248m River bottoms, marshes
    Valley 0.15-0.35 248-378m River valleys
    Plain 0.35-0.55 378-508m Typical steppe
    Upland 0.55-0.70 508-605m Rolling hills
    Highland 0.70-0.85 605-703m High terrain
    Hilltop 0.85+ 703-800m Exposed peaks 
    Meters = 150 + elevation × 650

    Generation¶

    Elevation is generated per local map tile using:

    elevation = blendedBase + (noise - 0.5) × biomeVariance + tileAdjustment

    Biome Variance¶

    Biome Variance Character
    Mountain Steppe 0.35 Most dramatic
    Forest-Steppe 0.20 Rolling
    River Valley 0.15 Moderate
    Pontic Steppe 0.12 Gentle
    Central Asian Steppe 0.12 Gentle
    Wetland 0.08 Flat
    Other 0.15 Default

    Tile Adjustments¶

    Tile Adjustment
    Hill +0.15
    Water / Marsh -0.15
    River Bank -0.10

    Local Slope Movement Cost¶

    slope = destinationElevation - currentElevation

if slope > 0 (uphill):
    cost × (1 + slope × 3.0)

if slope < 0 (downhill):
    cost × (1 + |slope| × 1.5)

    Steep uphill (slope 0.3): cost × 1.9. Downhill is cheaper but not free.

    Overworld Elevation Cost¶

    if elevDiff > 0 (uphill):   cost × (1.0 + elevDiff × 2.0)
if elevDiff < 0 (downhill): cost × Max(0.7, 1.0 - |elevDiff| × 1.0)

    Vision Bonus¶

    Elevation Bonus
    > 0.7 (Highland+) +2 tiles
    > 0.5 (Upland+) +1 tile

    Applies to both player and NPC vision.

    Rendering¶

    Tile color is multiplied by 0.85 + elevation × 0.30: - Low elevation (0.0): darker (× 0.85) - High elevation (1.0): brighter (× 1.15)
