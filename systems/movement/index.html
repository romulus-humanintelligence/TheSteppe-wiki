    Movement System¶

    Movement on both local maps and the overworld, on foot and mounted.

    Local Map Movement¶

    Each step costs 2 base minutes, modified by speed and terrain.

    Speed Calculation¶

    CurrentMph = BaseMph
CurrentMph /= SpeedStatModifier        — Max(0.5, 1.5 - SpeedStat × 0.05)
CurrentMph /= WoundMoveCostMultiplier  — leg/foot injuries
CurrentMph × EncumbranceModifier       — if carrying > 80% capacity
CurrentMph × SneakModifier             — 0.5x while sneaking
CurrentMph = Max(0.1, CurrentMph)

MoveCost = Max(1, 2 × ReferenceSpeed / CurrentMph)

    Encumbrance¶

    if CarryRatio > 0.8:
    overburden = (CarryRatio - 0.8) / 0.2
    speed × Max(0.5, 1.0 - overburden × 0.3)

    At 100% carry capacity, speed is reduced by 30%.

    Weather Movement Cost¶

    Weather Multiplier
    Clear / Cloudy / Rain / Fog 1.0x
    Heavy Rain 1.3x
    Snow 1.25x
    Blizzard 1.6x
    Dust Storm 1.2x

    Slope Cost¶

    See Altitude System for uphill/downhill multipliers.

    Multi-Tile Movement¶

    Fast entities (high Speed stat, mounted) can move multiple tiles per turn via an accumulator:

    MoveAccumulator += CurrentMph / ReferenceSpeed (3.0)
MaxTilesPerTurn = 5.0
Each 1.0 accumulator = 1 tile of movement

    Overworld Movement¶

    Overworld travel costs vary by biome:

    Biome Base Cost (min)
    Pontic Steppe 120
    Central Asian Steppe 120
    Sea Coast 120
    River Valley 100
    Forest-Steppe 150
    Desert Edge 160
    Taiga Fringe 180
    Mountain Steppe 200
    Wetland 200

    River crossing: +60 minutes per tile with river.

    Overworld Elevation Cost¶

    if elevDiff > 0 (uphill):   cost × (1.0 + elevDiff × 2.0)
if elevDiff < 0 (downhill): cost × Max(0.7, 1.0 - |elevDiff| × 1.0)

    Overworld Wind Effects¶

    Only applies when wind is Moderate or Strong:

    Direction Moderate Strong
    Headwind +15% +25%
    Tailwind -10% -15%
    Crosswind 0% 0%

    Mounted Movement¶

    See Horseriding for mounted speed formula.

    Horse base speed is 8.0 mph (vs 3.0 walking), modified by:

    • Horseriding skill factor (0.5-1.2x)
    • Horse energy factor (0.5-1.0x)
    • Bond factor (0.9-1.1x)

    Interactions¶

    • Movement generates sweat (0.5 per action, 0.25 sneaking, 0.15 mounted)
    • Movement costs energy (0.5 per action)
    • Leg/foot wounds add 0.15-0.80x cost penalty
    • Fractures add 0.5x per fractured leg/foot
